Compendia (In Progress)

React / Django / Python / MongoDB / HTML5 / Sass

Compendia is an in-progress solution to several problems that I ran into with collecting comic books. I always wished I could have an app that I could see which comics I owned, which ones I wanted, and which new releases were waiting for me at the local comic shop each week all in a user interface that was simple and usable. For this web app, I made a RESTful API using Django Rest Framework to serve the comic data from the database to any user who has an account. This is a real full-stack application that I designed, architected, and developed all myself.
